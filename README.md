# Student Attendance Tracker - Deploy Agent

## How to run the script
1. Place all source files in the same folder as setup_project.sh
2. Run: bash setup_project.sh
3. Enter a project name when prompted
4. Choose whether to update attendance thresholds

## How to trigger the archive feature
- Press Ctrl+C at any point while the script is running
- The script will automatically bundle the current state into a .tar.gz archive
- The incomplete directory will be deleted automatically
